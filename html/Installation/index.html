Pré-requisitos

Antes de começar, você vai precisar de algumas coisas:

  • Seu IDE ou editor de texto favorito
  • Python 3.7 - Python 3.9
  • PIP

Se ainda não o fez, reserve alguns minutos para ler os conceitos principais para entender o modelo de fluxo de dados do Streamlit.

Configure seu ambiente virtual

Independentemente da ferramenta de gerenciamento de pacotes que você está usando, recomendamos executar os comandos desta página em um ambiente virtual. Isso garante que as dependências extraídas do Streamlit não afetem nenhum outro projeto Python em que você esteja trabalhando.

Abaixo estão algumas ferramentas que você pode usar para gerenciamento de ambiente:

  • pipenv
  • poesia
  • venv
  • ambiente virtual
  • conda

Instale o Streamlit no Windows

O gerenciador de ambiente com suporte oficial do Streamlit no Windows é o Anaconda Navigator .

Instalar o Anaconda

Se você ainda não instalou o Anaconda, siga as etapas fornecidas na página de instalação do Anaconda .

Crie um novo ambiente com Streamlit

Em seguida, você precisará configurar seu ambiente.

  1. Siga as etapas fornecidas pelo Anaconda para configurar e gerenciar seu ambiente usando o Anaconda Navigator.

  2. Selecione o ícone "▶" ao lado do seu novo ambiente. Em seguida, selecione "Abrir terminal":

  3. No terminal que aparece, digite: 

    pip install streamlit

  4. Teste se a instalação funcionou: 

    streamlit hello

    O aplicativo Hello da Streamlit deve aparecer em uma nova aba no seu navegador!

Use seu novo ambiente

  1. No Anaconda Navigator, abra um terminal em seu ambiente (veja a etapa 2 acima).

  2. No terminal que aparece, use o Streamlit como de costume: 

    streamlit run myfile.py

Instale o Streamlit no macOS/Linux

O gerenciador de ambiente com suporte oficial do Streamlit para macOS e Linux é o Pipenv . Veja as instruções sobre como instalá-lo e usá-lo abaixo.

Instalar Pipenv

  1. Instale pip .

    Em um macOS: 

    sudo easy_install pip

    No Ubuntu com Python 3: 

    sudo apt-get install python3-pip

    Para outras distribuições do Linux, consulte Como instalar o PIP para Python .

  2. Instale pipenv . 

    pip3 install pipenv

Crie um novo ambiente com Streamlit

  1. Navegue até a pasta do seu projeto: 

    cd myproject

  2. Crie um novo ambiente Pipenv nessa pasta e ative esse ambiente: 

    pipenv shell

    Ao executar o comando acima, um arquivo chamado Pipfile aparecerá no formato myprojects/ . Este arquivo é onde seu ambiente Pipenv e suas dependências são declarados.

  3. Instale o Streamlit em seu ambiente: 

    pip install streamlit

    Ou se você quiser criar um ambiente facilmente reproduzível, substitua pip sempre pipenv que instalar algo: 

    pipenv install streamlit

  4. Teste se a instalação funcionou: 

    streamlit hello

    O aplicativo Hello da Streamlit deve aparecer em uma nova aba no seu navegador!

Use seu novo ambiente

  1. Sempre que você quiser usar o novo ambiente, primeiro você precisa ir para a pasta do seu projeto (onde o Pipenv arquivo está) e executar: 

    pipenv shell

  2. Agora você pode usar Python e Streamlit como de costume: 

    streamlit run myfile.py

    Para parar o servidor Streamlit, pressione ctrl-C .

  3. Quando você terminar de usar este ambiente, apenas digite exit ou pressione ctrl-D para retornar ao seu shell normal.
